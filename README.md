## 🧰 Basic Use

After setup, you can start with these tasks:

### Send funds
1. Open wallet.
2. Choose the asset you want to send.
3. Select Send.
4. Paste the address.
5. Enter the amount.
6. Check the network and fee.
7. Confirm the transfer.

### Receive funds
1. Open wallet.
2. Choose the asset you want to receive.
3. Select Receive.
4. Copy your address or scan the QR code.
5. Send that address to the other person or exchange.

### Check your history
1. Open the asset page.
2. Find the list of past transfers.
3. Review the date, amount, and status.
4. Match it with the address or transaction ID if needed.

### Add more accounts
1. Open settings or account management.
2. Choose Add account or Import account.
3. Pick the chain you want.
4. Follow the prompts.

## 🛡️ Security Tips

wallet handles private keys and funds, so use safe habits from the start.

- Keep your recovery phrase offline
- Use a long password
- Lock your PC when you leave it
- Do not share your seed phrase
- Check addresses before sending
- Use only trusted downloads
- Keep Windows updated
- Back up your wallet details in a safe place

If someone gets your recovery phrase, they can access your wallet. Treat it like cash and store it with care.

## 🧩 Common Windows Issues

If the app does not open, try these steps.

### The file will not start
- Right-click the app and choose Run as administrator
- Make sure the file finished downloading
- Re-download the file from the GitHub page

### Windows blocks the app
- Open the file again
- Choose More info if Windows shows it
- Select Run anyway if you trust the source

### The ZIP file will not extract
- Check that you have enough free space
- Try the built-in Windows extractor
- Download the file again if it looks damaged

### The wallet does not sync
- Check your internet connection
- Wait a few minutes and try again
- Close and reopen the app
- Make sure the correct network is selected

### The balance looks wrong
- Refresh the app
- Check whether you are on the right chain
- Confirm that the address matches your account
- Look at the transaction history for pending transfers

- **Transaction**: A transfer on the blockchain
- **Fee**: The cost to process a transfer
